* The game is currently have the following settings: 
(can be modified in the canvas node in cocos creator)
	* time limit: 60s
	* number of bots: 3
	* number of eggs appear on screen at once: 1
	* bot speed is set to randomized at spawn (with the max speed still being
	lower than the player's speed)

* Editor version is 3.1.1
* Physics setting is chosen to be Builtin 2D Physics Systems. 
Using other Physics may disrupt some of the game's functionality
